# Model AI EcoLens

Letakkan model YOLO custom di folder ini dengan nama:

```text
yolov8_waste.pt
```

Jika file tersebut belum tersedia, aplikasi akan memakai `yolov8n.pt` sebagai YOLOv8 pretrained untuk demo. Bila YOLO belum bisa dipakai, aplikasi tetap dapat berjalan memakai analisis visual fallback agar demo tidak terhenti.
